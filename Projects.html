Core Projects


Project Title: Automatic Targeting Military Radar Vehicle.

The "Automatic Targeting Military Radar Vehicle" is a semi-autonomous vehicle designed to enhance soldier safety by detecting and neutralizing enemy vehicles using radar and a camera system. It operates in both manual and autonomous modes, controlled via Bluetooth with an ESP32 interface for live targeting. In manual mode, the operator can control the vehicle and lock onto threats, while in autonomous mode, it navigates obstacles and engages targets independently. Powered by Arduino and featuring a Java-based control interface, the vehicle is versatile for military operations, surveillance, and even disaster response. Key challenges include managing signal interference and ensuring real-time processing for quick decision-making.

Project Title: Digital Announcement Board Using IoT.

This project leverages the ESP8266 NodeMCU module to create an IoT-based digital announcement system. The board consists of four LCD I2C displays (16x2) that can be remotely managed through a web server. Each display is assigned a unique IP address, enabling real-time, targeted announcements for different sections of an organization. With applications in educational institutions, corporate offices, hospitals, and restaurants, the system improves communication and content dissemination. Future upgrades may include voice control and interactive touch screens, further enhancing the efficiency and engagement of the system.

Project Title: Traffic Management System.


The Traffic Management System is part of a comprehensive approach to address traffic issues using machine learning and IoT. This specific phase focuses on implementing predictive analytics to forecast traffic congestion and optimize road usage. The project leverages real-time data from sensors, which is processed using a neural network model to predict traffic patterns and suggest optimal routes. It integrates IBM cloud services for efficient data storage, processing, and visualization. The solution aims to reduce traffic congestion and enhance public safety by providing accurate traffic insights. Additionally, it offers a user-friendly interface for real-time updates on traffic conditions and expected delays. This traffic management system has practical applications in urban environments, where dynamic traffic data can support route optimization and infrastructure planning.
